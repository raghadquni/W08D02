# Backend project
Register/Login project with authentication and authorization

# Packages used 

* **express**

To bulid Server
``` 
npm i express
```
* **dotenv**

To hide sensitive information
``` 
npm i dotenv
```

* **nodemon**

To Refresh the server automatically
``` 
npm i nodemon
```

* **mongoose**

To save our database and the possibility to use it

``` 
npm i mongoose
```
